John Evans Dakurah - Academic PhD Profile Website

This is a plain static website. No React, no npm install, and no build step required.

How to view:
1. Open this folder.
2. Double-click index.html.

Files:
- index.html: main website content
- styles.css: website styling
- script.js: mobile menu and footer year
- assets/john-evans-dakurah.jpg: profile photo
- assets/John_Evans_Dakurah_Resume.pdf: downloadable resume PDF

Deployment note:
Upload index.html, styles.css, script.js, README.txt, and the full assets folder to GitHub Pages. The Download Resume button will work as long as the PDF remains inside the assets folder.


SEO files added:
- robots.txt
- sitemap.xml
- structured data in index.html
- Open Graph and Twitter metadata
- favicon.svg

For GitHub Pages, upload index.html, styles.css, script.js, robots.txt, sitemap.xml, and the full assets folder.
After deployment, submit sitemap.xml in Google Search Console.

SEO note: canonical page URL is set to https://johnevansdakurah.github.io/ for the static GitHub Pages user site.
